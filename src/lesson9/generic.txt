Обобщённое программирование — это такой подход к описанию данных и алгоритмов,
который позволяет их использовать с различными типами данных без изменения их описания.

generics (дженерики) или <<контейнеры типа T>> — подмножество обобщённого программирования


Создание generic класса:

public class GenericClass<T> {}
После имени класса пишется <Т>, "Т" может использоваться внутри класса (для указания типа).
Т – это тип, который должен быть определён позже (при создании объекта класса).

Внутри класса - T в объявлении поля:
public class GenericClass<T> {
    private T val; - объявляется переменная дженерик-типа (generic type), т.о. её тип будет указан позже, при создании объекта класса
}

По аналогии с дженерик-классами, можно создавать универсальные методы (дженерик-методы),
то есть методы, которые принимают общие типы параметров (это не мотоды дженерик класса).
Универсальные методы удобны, когда одна и та же функциональность должна применяться к различным типам.

public static <T> void genericVoid(List<T> list, T val) {
        for (int i = 0; i < list.size(); i++)
            list.set(i, val);
    }
